# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately using GitHub's Security Advisories feature:

**[Report a vulnerability](https://github.com/DownAtTheBottomOfTheMoleHole/yarn-ado/security/advisories)**

Do **not** disclose security issues publicly until they have been reviewed and patched.

- A response will be made as quickly as possible.
- After the vulnerability is addressed, the fix will be documented in [CHANGELOG.md](CHANGELOG.md).
